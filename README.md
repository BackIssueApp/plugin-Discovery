# Discover

A browsable feed of new & notable comics for BackIssue, driven by ComicVine:
**New releases**, **Latest additions**, personalized **For you** picks based
on what's in your library, and **Popular** series — each with one-click add
to your collection.

## Install

One click from **Sidebar → Plugins** in BackIssue, or drop this folder into
the app's `plugins/` directory and restart.

## Setup

Nothing required — a **Discover** entry appears in the sidebar (only for
users who can add to the library). It uses the same ComicVine/metadata
configuration as the rest of the app.

Full guide: <https://backissue.app/discover>
